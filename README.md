Linux Commands
1. mkdir

Usage:

mkdir <directory_name>


Purpose: Creates a new directory.

2. touch

Usage:

touch <file_name>


Purpose: Creates an empty file.

3. mv

Usage:

mv old_name new_name


Purpose: Renames or moves a file/directory.

4. cat

Usage:

cat <file_name>


Purpose: Displays the content of a file.

5. head

Usage:

head -n 5 <file_name>


Purpose: Displays the first 5 lines of a file.

6. tail

Usage:

tail -n 5 <file_name>


Purpose: Displays the last 5 lines of a file.
